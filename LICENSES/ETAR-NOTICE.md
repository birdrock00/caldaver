# Etar Calendar attribution

Caldaver's mobile Calendar interface includes and adapts source code and user
interface concepts from [Etar Calendar](https://github.com/Etar-Group/Etar-Calendar).
The reference source used for this work was Etar commit
`db1dff8831553ef8126b72c268b3c8f9a57b355b`. The adapted implementation has
been changed to run inside Caldaver's responsive web and Capacitor interfaces
and to use Caldaver's existing CalDAV calendar data and calendar-selection UI.
It is not an unmodified Etar application and is not endorsed by the Etar
Project.
